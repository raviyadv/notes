# Git commands

>First time initializing, add, commit and push
```
echo "# notes" >> README.md
git init
git add README.md
git commit -m "first commit"
git remote add origin https://github.com/raviyadv/notes.git
git push -u origin master
```
>Or push an existing repository from the command line
```
git remote add origin https://github.com/raviyadv/notes.git
git push -u origin master
```
